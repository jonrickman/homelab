## Youtube Utilities
#### Goal:  
`For educational purposes only.  
This application will be able to download videos to convert them into mp4s through the use of ffmpeg. After files are in mp4 form then they will be converted into a text format through speech-to-text. The contents of the text will then be stored in a database and querable.  `

#### Notes:
`Completely not intended to be a production nor scalable application. Design choices made do not necessarily reflect most optimum choices though those specifics will be talked about in the individual readme in yt folder.`
